# 👋 はじめまして、金昇宇と申します。

バックエンドエンジニアとしてキャリアを積みつつ、<br＞
フロントエンドでは JavaScript / CSS / HTMLを用いた画面開発に3年携わってきました。<br＞<br＞

現在は、よりモダンなフロントエンド技術（React / Next.js / TypeScript）を<br＞
体系的に学び、フルスタックエンジニアとしてのスキル拡張に取り組んでいます。<br＞<br＞

Udemy の実践的な講座を通じて、実際に手を動かしながら<br＞
基礎から応用まで幅広く理解を深めてきました。<br＞
本ページでは、その過程で作成したアプリケーションや、<br＞
学習内容をまとめています。<br＞
